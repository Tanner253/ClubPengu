### Problem

- Limited customization options

- Didn't allow for promo code gating

### Solution
- Removed automatic wizard hat assignment for all dogs
- Created new promo code "WZRDOG" that unlocks wizard hat cosmetic

- Wizard hat now only appears if user has it unlocked and equipped

### Changes

- **"DOG" code**: Now only unlocks dog character type (no cosmetics)

  - Unlocks only the wizard hat, not the character

- `PlayerBuilder.js`: Dogs now only get wizard hat if `data.hat === 'wizardHat'`
