# Slot Machine Meme Symbols

Drop your meme images here to use them on the casino lobby slot reels.

## How to add images

1. Save PNG or WebP files in this folder (`public/slot-symbols/`)
2. Use the filenames listed in `src/config/slotSymbols.js`, or add new entries there
3. Refresh the game — reels will pick them up automatically

## Recommended specs

- **Format:** PNG or WebP (transparent background works great)
- **Size:** 256×256 or 512×512 px
- **Style:** Square or near-square memes (trollface, pepe, wojak, doge, etc.)

## Default filenames

| File | Meme |
|------|------|
| `trollface.png` | Trollface |
| `pepe.png` | Pepe |
| `wojak.png` | Wojak |
| `doge.png` | Doge |
| `chad.png` | Chad |
| `stonks.png` | Stonks |
| `gigachad.png` | Gigachad |
| `crab.png` | Crab rave / whatever you want |

Missing files show a placeholder until you add the image.
