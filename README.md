## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Tanner253/waddlebet.git
   cd waddlebet
   ```

2. **Install dependencies for the game**
   ```bash
   cd waddlebet
   npm install
   ```

3. **Run the game demo**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) in your browser.

4. **Run the server (for multiplayer)**
   ```bash
   npm run dev:server
   ```

5. **Run both client and server**
   ```bash
   npm run dev:all
   ```

### Running the Whitepaper Site

1. **Navigate to whitepaper folder** (from repo root)
   ```bash
   cd whitepaper
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.
